A user must be able to:
upload csv data
choose a graph type and display that graph type on a html.
The user must be able to switch between graph types for the uploaded data

Process
input data
select type
create chart
    - convert correct values for type from data
    - display the correct type of graph on the html page

Lemming Charts:
A lemming chart is made out of a lemming object.
A lemming chart is a visual presentation on an HTML page of a custom matplotlib graph.

A lemming object has:
- Data sets (csv, txt, xml)

Chart Type Variables:
- Line: [x, y]
- Bar: ["keys", "values"]
- Scatter: [x, y]
- Histogram: [12, 15, 12, 13, 18, 15]
- Pie: ["keys", "values"]

ConvertDataToType
This gets the correct type variables from the uploaded data
this way it can be used to display on the page

Graph Types:
- Line
```python
plt.plot(x, y, label="Trend")
plt.title("Line Plot")
plt.xlabel("X-axis")
plt.ylabel("Y-axis")
plt.legend()
plt.show()
```

- Bar
```python
plt.bar(categories, values, color='skyblue')
plt.title("Bar Chart")
plt.xlabel("Categories")
plt.ylabel("Values")
plt.show()
```

- Scatter
```python
plt.scatter(x, y, color='purple')
plt.title("Scatter Plot")
plt.xlabel("X-axis")
plt.ylabel("Y-axis")
plt.show()
```

- Histogram
```python
plt.hist(data, bins=5, color='orange', edgecolor='black')
plt.title("Histogram")
plt.xlabel("Bins")
plt.ylabel("Frequency")
plt.show()
```

- Pie
```python
plt.pie(sizes, labels=labels, autopct='%1.1f%%', startangle=140)
plt.title("Pie Chart")
plt.show()
```

A lemming chart displayed in the HTML is a custom presentation of the standard matplotlib graph.

Classes:
- LemmingChart
- LemmingObject

Usage:
```python
lemming_object = LemmingObject(dataset_path='path/to/dataset.csv', chart_type='Line', chart_params=['x', 'y'])
lemming_chart = LemmingChart(lemming_object)
html_snippet = lemming_chart.render_html()
print(html_snippet)
